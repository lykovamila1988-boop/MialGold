# MialGold

Local MILA Office codebase for AI-assisted content, analytics, and business operations.

## Main apps

- `mila-office/` - Flask web UI and multi-agent office runtime.
- `mila-agent/` - standalone local marketing agent.
- `tools/` - Instagram, Threads, Gumroad, Supabase, and reporting helper scripts.

## Secrets

Do not commit real `.env` files. Use `env.template` and `.env.example` files instead.
