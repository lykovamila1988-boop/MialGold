# Перенаправление Васи (Планировщика) дальше

## Кто такой Вася?
**Вася** — планировщик контента и публикаций. Он:
- Создаёт расписание публикаций
- Проверяет что готово к публикации
- Ставит посты в очередь
- Напоминает что нужно снять/написать
- Ведёт лог публикаций

---

## ✅ NEXT ACTIONS для Васи

Когда Вася завершит работу, он может передать результат одному из трёх агентов:

### 1️⃣ **Передать Manager** (Развернуть в задачи)
**Когда:** Вася создал план/расписание, но нужно разбить на конкретные задачи

**Действие:**
```
Кнопка: "Развернуть в задачи"
Передать: Manager
Промпт: "Разложи этот план в задачи для офиса: кто, что, когда, критерий готовности"
Результат: Task-list с владельцами и дедлайнами
```

**Типичный сценарий:**
- Вася создал месячный календарь публикаций
- Manager разбивает на еженедельные задачи
- Каждому агенту/сотруднику назначается что делать

---

### 2️⃣ **Передать Producer** (Согласовать план)
**Когда:** Вася готовит расписание, нужна валидация

**Действие:**
```
Кнопка: "Согласовать с продюсером"
Передать: Producer
Промпт: "Проверь этот календарь как продюсер: реалистичность, приоритеты, зависимости, риски"
Результат: Утвержденный (или скорректированный) календарь
```

**Типичный сценарий:**
- Вася предложил запустить 3 рила на неделю
- Producer проверяет: хватит ли времени на съёмку?
- Producer утверждает или просит отсрочку

---

### 3️⃣ **Передать Tyoma** (Опубликовать в Telegram канал)
**Когда:** Готов контент для публикации в канал для КЛИЕНТОВ

**Действие:**
```
Кнопка: "Опубликовать в Telegram"
Передать: Tyoma (Telegram-менеджер)
Промпт: "Опубликуй этот контент в Telegram канал для подписчиков"
Результат: Сообщение в Telegram-канале @liudmyla.lykova
```

**Типичный сценарий:**
- Marina создала контент про отношения
- Tyoma публикует в Telegram-КАНАЛ (для клиентов):
  ```
  Когда ты выбираешь партнёра из страха потери...
  
  Это не любовь.
  Это защита от одиночества.
  
  Любовь выбирает из целостности.
  ```
- Это часть воронки: подписчики → прогрев → покупка практикума → консультация

## 🚀 БЫСТРАЯ ИНСТРУКЦИЯ

### Вася говорит: "Я готовый план"
**→ Выбери куда передать:**

| Нужно | Агент | Кнопка | Результат |
|------|-------|--------|-----------|
| Разбить на задачи | Manager | Развернуть в задачи | Task-list для команды |
| Проверить реалистичность | Producer | Согласовать с продюсером | Утвержденный план |
| Опубликовать клиентам | Tyoma | Опубликовать в Telegram | Пост в канале @liudmyla.lykova |

### Вася говорит: "Мне нужны конкретные задачи"
**→ Передай Manager**
- Действие: "Развернуть в задачи"
- Manager разобьёт план на еженедельные задачи с дедлайнами

### Вася говорит: "Может ли это реально сделать за месяц?"
**→ Передай Producer**
- Действие: "Согласовать с продюсером"
- Producer проверит реалистичность и одобрит/скорректирует

### Вася говорит: "Готов контент, нужно опубликовать"
**→ Передай Tyoma**
- Действие: "Опубликовать в Telegram"
- Tyoma опубликует в **Telegram-КАНАЛ для КЛИЕНТОВ** (воронка продаж)
- Это НЕ сообщение команде, а контент для подписчиков

## 📋 QUICK COMMANDS (Вася)

Когда ты в чате с Васей, можешь использовать эти команды:

```
/график         → Создай расписание на неделю
/очередь        → Покажи очередь публикаций
/напомни        → Что нужно снять/написать?
/логи           → Все публикации за месяц
/помощь         → Помощь от Васи
/выход          → Выход из чата
```

## 🔄 ПРОЦЕСС РАБОТЫ С ВАСЕЙ

```
1. ТЫ даёшь Васе задачу
   └─ "Создай план публикаций на месяц"

2. ВАСЯ создаёт расписание
   └─ "Пн-Пт в 10:00, Вт/Чт/Сб - рилсы, каждую пятницу оффер"

3. ТЫ ВЫБИРАЕШЬ NEXT ACTION
   ├─ Manager: "Развернуть в задачи"
   ├─ Producer: "Согласовать с продюсером"
   └─ Tyoma: "Опубликовать в Telegram"

4. ВЫБРАННЫЙ АГЕНТ продолжает работу
   └─ "Делаю то-то..."

5. РЕЗУЛЬТАТ передаётся дальше по цепочке
   └─ Или завершается
```
