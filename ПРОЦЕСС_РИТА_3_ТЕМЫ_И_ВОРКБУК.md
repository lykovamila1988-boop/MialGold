# Процесс: Rita → 3 Темы → Скачать Воркбук для GAMMA

## ШАГ 1: Rita анализирует аудиторию и предлагает TOP-3 ТЕМЫ

Rita (Product Architect) работает в двух режимах:

### Режим А: АНАЛИЗ АУДИТОРИИ → ТЕМЫ

```
Rita.analyze_audience() 
  ├─ Смотрит топ-посты по вовлечённости (что заходит)
  ├─ Анализирует боли в комментариях и лидах
  ├─ Прочитает экспертные темы профиля
  └─ Предлагает TOP-3 ТЕМЫ для воркбука
```

**Каждая тема включает:**
- Боль аудитории (на реальных данных, с цитатой)
- Обещание результата
- Связь с воронкой ($37→$120→пакеты)
- Черновое название

**Пример вывода Rita:**

```
TOP-3 ТЕМЫ ДЛЯ ВОРКБУКА:

1️⃣  ТЕМА: "Как узнать свой паттерн в отношениях"
    Боль: В комментариях 40% женщин пишут "это про мене"
    Данные: пост про паттерны Спасателя 2.3K лайков
    Обещание: За 5 дней узнаешь свой паттерн
    Продажа: $37 воркбук → $120 консультация → диагностика
    
2️⃣  ТЕМА: "Почему я выбираю не тех мужчин"
    Боль: Топ-комментарий: "я 5 раз выходила и все одинаковые"
    Данные: post_reach=4200, comments=127
    Обещание: Поймёшь настоящую причину и получишь инструмент выбора
    Продажа: Прямой путь к пакету $420 (4 консультации)
    
3️⃣  ТЕМА: "От кого я прячусь в себе"
    Боль: Часто появляется в stories вопросах
    Данные: Теория + конкретные кейсы лидов
    Обещание: Поймешь корень тревожной привязанности
    Продажа: Фундамент для $750 пакета (12 консультаций)
```

## ШАГ 2: Режим Б - Rita создает СТРУКТУРУ ПРОДУКТА

Когда одна из 3 тем выбрана → Rita выдает структуру:

```
СТРУКТУРА ПРОДУКТА:

1. Название продукта
2. Для кого и какое обещание результата
3. Структура: главы/модули в логическом порядке
4. Упражнения и рабочие страницы
5. Тон и ограничения (голос Людмилы)
6. ТЗ для Марины (что писать)
```

**Пример:**

```
НАЗВАНИЕ: "Почему я снова выбрала не того"

ДЛЯ КОГО: Женщины 25-45, болезненные отношения, тревожная привязанность

ОБЕЩАНИЕ: За 30 дней через 4 шага осознания ты поймешь свой паттерн
и начнешь выбирать иначе

СТРУКТУРА (36 страниц):
├─ ОБЛОЖКА (1 стр)
├─ ЧТО ТЫ ПОЛУЧИШЬ (1 стр)
├─ ШАГ 1: ОСОЗНАНИЕ (8-10 стр)
│  Мини-тесты на паттерны + истории
├─ ШАГ 2: ПОНИМАНИЕ (8-10 стр)
│  Корни в семье + ранние отношения
├─ ШАГ 3: ИСТОРИИ (8-10 стр)
│  Видишь себя в других + примеры
└─ ШАГ 4: ВЫБОР (8-10 стр)
   Инструменты нового выбора + план

УПРАЖНЕНИЯ:
├─ Мини-квиз "Узнай свой паттерн"
├─ "Мои истории с мужчинами" (анализ 3 отношений)
├─ Дневник наблюдений (7 дней)
└─ План действий на месяц

ТОН: Теплая подруга-психолог. "Ты". Без диагнозов, без обвинений.
ОГРАНИЧЕНИЯ: Не про травму, не про детство родителей, не про абьюз
```
