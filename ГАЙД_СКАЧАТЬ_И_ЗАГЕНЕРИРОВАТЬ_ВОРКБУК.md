## ШАГ 3: Сгенерировать воркбук (детальная инструкция)

### Структура воркбука (из gamma_prompt):

```
БЛОК 1: DESIGN INSTRUCTIONS (вставляй всегда в начало)
├─ Color palette (цвета)
├─ Typography (шрифты, размеры)
├─ Layout (отступы, сетка, формат A4)
├─ Navigation bar (навигация между разделами)
└─ Page numbering (нумерация страниц)

БЛОК 2: КОНТЕНТ (вставляй по частям)
├─ Слайд 1: Обложка
├─ Слайд 2: Структура / Что ты получишь
├─ Слайд 3-X: Основной контент
└─ ... и т.д.
```

### Шаги генерации:

**Шаг 1:** Первый запрос
```
[вставь полный блок DESIGN INSTRUCTIONS]
[вставь слайды 1-5 (обложка + структура + начало контента)]
```
→ Нажми Generate → Скачай результат

**Шаг 2:** Второй запрос (дополнение)
```
[вставь DESIGN INSTRUCTIONS снова]
[вставь слайды 6-15 (следующий блок контента)]
```

**Шаг 3:** Продолжай так для всех 36 страниц
