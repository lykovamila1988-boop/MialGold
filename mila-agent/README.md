# MILA Agent — Локальный маркетинг-агент

Персональный ИИ-агент для @liudmyla.lykova.
Читает файлы из `E:\MILA GOLD`, пишет посты, анализирует Instagram.

---

## Быстрый старт (5 минут)

### 1. Установить Python
Скачать с python.org — версия 3.10 или выше.

### 2. Установить зависимости
```bash
cd path/to/mila-agent
pip install -r requirements.txt
```

### 3. Настроить .env
Скопировать `env.template` → `E:\MILA GOLD\.env`
Заполнить ключи:
- `ANTHROPIC_API_KEY` — с console.anthropic.com
- `INSTAGRAM_ACCESS_TOKEN` — из Facebook Developer
- `INSTAGRAM_BUSINESS_ACCOUNT_ID` — ID бизнес-аккаунта

### 4. Запустить
```bash
python agent.py
```

## Команды

| Команда | Что делает |
|---------|-----------|
| `/аналитика` | Статистика Instagram за неделю |
| `/комменты` | Читает комментарии → пишет ответы |
| `/контент` | Создаёт 5 постов и сохраняет в папку |
| `/reels` | 3 идеи для вирусных Reels |
| `/файлы` | Показывает содержимое папки |
| `/dm` | Читает Direct Messages |
| `/помощь` | Список всех команд |
| `/выход` | Завершить |

Или просто напиши что нужно на русском языке.

## Структура папки E:\MILA GOLD

```
E:\MILA GOLD\
  .env                    ← ключи API (секрет!)
  content\
    posts\                ← тексты постов (создаёт агент)
    reels\                ← сценарии Reels
    stories\              ← сторис
  analytics\              ← отчёты (создаёт агент)
  03-clients\             ← клиентские материалы
  04-telegram\            ← Telegram контент
  logs\
    published.log         ← история публикаций
```
