CUSTOMER JOURNEY STAGES
================================================================================

Stage 1: ЛИДИРОВАНИЕ (Intake Form)
  - Source: Instagram DM, Telegram, website form
  - From: Лера (sales)
  - Alina: reads intake → identifies pattern → creates profile → logs intake stage
  - Output: Client profile ready for consultation

Stage 2: КОНСУЛЬТАЦИЯ/ДИАГНОСТИКА (Consultation)
  - Types: free diagnosis (20 min) or paid consultation ($120)
  - Conductor: Людмила
  - Alina: receives session notes → structures insights → updates profile
  - Output: Updated profile with key insights, blocks, motivation level

Stage 3: ПАКЕТ СЕССИЙ (Package of Sessions)
  - Types: 4-session package ($420) or 8-session package ($750)
  - Alina's role:
    * Before each session: reads profile, prepares Людмила with context
    * After each session: structures notes, updates progress, logs completion
    * Monitors adherence: tracks completed sessions, identifies risks
  - Output: Continuous client journey tracking

Stage 4: FOLLOW-UP & REPEAT (Maintenance & Repeat Purchases)
  - Timeline: 2 weeks, 1 month, 3 months post-package
  - Alina: checks stability, suggests follow-up or repeat package
  - Recommendation: практикум (workbook) for independent work
  - Output: LTV calculation, repeat customer tracking

================================================================================
CHAIN_ID TRACKING
================================================================================

Each customer interaction is tracked with unique chain_id:

  Format: journey_20260608_143015_abc123
          ├─ journey (prefix)
          ├─ 20260608_143015 (timestamp)
          └─ abc123 (random suffix)

Usage:
  1. Extract from incoming message: [chain_id: journey_...]
  2. Log in client_journey.log with every milestone
  3. Log in chain.log with agent transitions
  4. Store in client profile for future reference

Enables:
  - Full customer journey audit trail
  - Link intake → consultation → package → repeat
  - Measure time between stages
  - Track which agent helped at each stage
  - Calculate LTV with full context

Example log:
  [2026-06-08 14:30] CLIENT=Анна STAGE=intake CHAIN=journey_20260608_143015_abc123
  [2026-06-08 14:31] agent=alina from=lera verdict=ready_next chain=journey_20260608_143015_abc123
  [2026-06-08 15:00] CLIENT=Анна STAGE=consultation CHAIN=journey_20260608_143015_abc123
