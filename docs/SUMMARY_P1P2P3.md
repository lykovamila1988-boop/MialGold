```
✅ P1_REELS_RECOMMENDATIONS_SETUP.md     (инструкция полная)
```

### Как работает:
1. **Каждый понедельник в 09:00 UTC** запускается анализ
2. **Анализирует Reels:**
   - Читает последний отчёт posts_*.json
   - Фильтрует только Reels (VIDEO)
   - Считает reach, likes, comments, engagement rate
3. **Генерирует рекомендации** через Claude API:
   - Какие темы работают лучше?
   - Какой формат выбрать?
   - 2-3 конкретные идеи для следующей недели
4. **Отправляет Марине** в Telegram
5. **Сохраняет отчёт** в папку reels-recommendations/

### Ожидаемый результат:
- ↑ 5-10% конверсии Reels в подписку/Telegram
- Лучшие посты определяются на основе данных, не интуиции

### Следующие шаги:
```bash
# 1. Проверить конфиг (ANTHROPIC_API_KEY нужен!)
cd tools && python reels_recommendations.py

# 2. Протестировать (без отправки Марине)

# 3. Отправить Марине
python reels_recommendations.py --send

# 4. Активировать в n8n (еженедельное расписание)
```
