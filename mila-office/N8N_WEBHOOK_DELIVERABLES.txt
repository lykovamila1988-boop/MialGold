================================================================================
QUICK START
================================================================================

1. Generate token:
   python -c "import secrets;print(secrets.token_urlsafe(32))"

2. Set in tools/.env:
   N8N_WEBHOOK_TOKEN=<generated>
   N8N_STATUS_WEBHOOK_URL=http://localhost:5678/webhook/office-status

3. Start service:
   cd mila-office && python n8n_webhook.py

4. Verify:
   curl http://127.0.0.1:5052/health

5. Test trigger:
   curl -X POST http://127.0.0.1:5052/api/n8n/trigger-chain \
     -H "Authorization: Bearer <TOKEN>" \
     -H "Content-Type: application/json" \
     -d '{"chain_config": {"chain_name": "new_client"}}'

================================================================================
TROUBLESHOOTING
================================================================================

Service won't start?
  - Check N8N_WEBHOOK_TOKEN is set in tools/.env

401 Unauthorized?
  - Verify token matches N8N_WEBHOOK_TOKEN
  - Check header format: "Bearer <TOKEN>"

Chain stuck in "pending"?
  - GET /api/n8n/chain/<id> to check
  - GET /api/n8n/logs/<id> to view events

Webhook not reaching n8n?
  - Check N8N_STATUS_WEBHOOK_URL is reachable
  - View n8n_webhook.log for failures
  - Verify n8n webhook listener is active

See N8N_WEBHOOK_API.md for detailed troubleshooting.
