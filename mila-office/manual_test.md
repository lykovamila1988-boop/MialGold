# 🧪 РУЧНОЕ ТЕСТИРОВАНИЕ ЦЕПОЧКИ

## Инструкции

Flask запущен на http://127.0.0.1:5000

### Тест 1: Проверка доступа
1. Откройте http://127.0.0.1:5000 в браузере
2. Должна загрузиться главная страница с агентами
3. ✅ Если загрузилось — Flask работает

---

## Тест 2: Цепочка передачи (Марина → Виктория)

### Шаг 1: Выберите Марину
- Нажмите на вкладку **Марина** (💼 иконка)

### Шаг 2: Напишите сообщение
В текстовое поле напишите:
```
Готовый пост про выбор в отношениях. [VERDICT: ready_next] [→ victoria]
```

**Объяснение служебных тегов:**
- `[VERDICT: ready_next]` — статус "готово для следующего агента"
- `[→ victoria]` — указание на следующего агента (Викторию)

### Шаг 3: Отправьте сообщение
- Нажмите кнопку **Отправить** (или Enter)
- Система покажет "Марина печатает…"
- Ждите 10-30 секунд пока Марина ответит

### Шаг 4: Смотрите результат
**ЧТО ДОЛЖНО ПРОИЗОЙТИ:**

✅ **ПРАВИЛЬНО (если цепочка работает):**
1. Марина ответит на ваше сообщение
2. **АВТОМАТИЧЕСКИ** откроется вкладка **Виктория** (без вашего клика!)
3. В истории Виктории будут видны:
   - Ваше сообщение для Марины
   - Ответ Марины
4. Виктория может редактировать текст

❌ **НЕПРАВИЛЬНО (если цепочка сломана):**
1. Марина ответит
2. Вкладка останется на Марине (не переключится)
3. Нужно ручно кликнуть на Викторию
4. История может быть потеряна

---

## Тест 3: Полная цепочка (опционально)

Если Тест 2 пройден, продолжите:

### 5. Виктория редактирует
- Виктория видит пост от Марины
- Отредактируйте или добавьте комментарий
- В ответе напишите: `[VERDICT: done]`
- Нажмите **Отправить**

### 6. Система покажет popup
**ЧТО ДОЛЖНО ПРОИЗОЙТИ:**
- Системное сообщение: "✅ Редактура завершена"
- Кнопка **"Скачать TXT"** чтобы загрузить готовый файл
