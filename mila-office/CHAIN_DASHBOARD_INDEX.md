## ❓ FAQ

**Q: Нужно ли устанавливать новые пакеты?**
A: Нет. Используются только Flask (уже есть) и стандартная библиотека.

**Q: Дашборд замедляет webapp.py?**
A: Нет. Это отдельный blueprint, не влияет на производительность.

**Q: Можно ли использовать без логирования?**
A: Да, но дашборд будет пуст. Логирование нужно для данных.

**Q: Где хранятся данные?**
A: В `mila-office/memory/events.jsonl` (JSONL файл, append-only).

**Q: На production что делать?**
A: Ротирование логов раз в месяц (архив старых событий).

**Q: Как экспортировать историю?**
A: Читайте JSON из events.jsonl или используйте API /chains/api/history.

**Q: Можно ли подключить Supabase?**
A: Да, мигрируйте события в таблицу chain_events (на future).
